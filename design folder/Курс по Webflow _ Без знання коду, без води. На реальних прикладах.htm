1 модуль

(6 уроків)

Знайомство з конструкторами, огляд webflow, основи html i css

  • Огляд популярних конструкторів, плюси і мінуси
  • Знайомство з інтерфейсом Webflow в деталях
  • Зрозумілою мовою про основи HTML та CSS, найменування, розміри, властивості Overflow, Fit
  • Одиниці вимірювання
  • Блокова модель, відступи, дисплей (Block, Inline, Inline-Block та None), основні елементи
  • Типографіка
  • Flex, grid та позиціонування (те, що потрібно для кращого розуміння побудови сайтів на конструкторі)

2 модуль

(9 уроків)

створюємо лендинг на webflow

  • Створення і налаштовання файлу для роботи
  • Створення головної секції (hero screen) сайту
  • Створення прикріпленого навігаційного меню (header), створення компоненту
  • Верстка всього сайту, створення карток.
  • Фоновий колір сторінки, Borders, Radius (заокруглення). Закріплюємо flex та grid на практиці
  • Створення footer сайту
  • Блок з підпискою на сайті, ID атрибут
  • Додаємо скрол на сторінці (якірні посилання)
  • Адаптивний дизайн для мобільних телефонів та планшетів
  • UI-kit
  • Переносимо файл із Figma

3 модуль

(4 уроки)

анімація і взаємодії

  • Створюємо плаваючу кнопку з функцією скролу нагору

FAQ

питаннячка

Чи потрібно мені знати код?

Для роботи з Webflow знання коду не потрібне. Все створюється за рахунок зручного інтерфейса. Проте ми розберемось з основами HTML та CSS для того, щоб робота у Webflow була для тебе зрозумілою та комфортною. А в одному модулі спробуємо навіть покодити.

Що я буду знати після курсу?

Після курсу ти зрозумієш основи розробки Webflow, вивчиш основи HTML та CSS, вивчиш інтерфейс Webflow, навчишся користуватись основними елементами, тегами та класами та налаштовувати елементи. Ти засвоїш всі знання на практиці та створиш повноцінний анімований лендинг, підключиш CMS колекції, навчишся створювати блогиз динамічними сторінкам і підключати оплати та сторонні сервіси. В кінці ми заанімуємо наш сайт та навіть трохи покодимо і підключимо кастомний слайдер.

В результаті ти отримаєш повноцінний перший лендинг, створений на Webflow, на якому ти засвоїш всі особливості платформи з урахуванням оновленого інтерфейсу.

Чи є підтримка на курсі?

Ні, це міні курс без підтримки, для самостійного вивчення. Але ми додали практичні завдання та тест на самоперевірку, щоб кожен студент міг закріпити свої нові знання. Уроки записані таким чином, щоб ти міг повторювати за нами та крок за кроком створити свій перший анімований та адаптивний лендинг.

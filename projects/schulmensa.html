Schulmensa

Десктопна система обліку та генерації рахунків-фактур для шкільних їдалень.

Electron React TypeScript SQLite3 Tailwind CSS Nodemailer SendGrid

Як все починалося (Сторi)

Шкільні їдальні щомісяця стикаються з паперовою рутиною: потрібно розрахувати вартість обідів для сотень учнів та розіслати рахунки батькам. Я розробив Schulmensa — надійний десктопний додаток, який автоматизує цей процес, створює PDF-рахунки та розсилає їх на email одним натисканням.

Виклики & Складні моменти

Обробка фінансових даних вимагає високої точності. Крім того, автоматична масова розсилка листів часто потрапляє у спам, якщо поштовий клієнт налаштований неправильно, а локальна база даних мала працювати безвідмовно.

Фінальна Перемога

Я побудував базу даних на базі SQLite3 з індексацією для швидкого пошуку. Реалізував подвійний міст відправки листів: через класичний `Nodemailer` (SMTP) для локальних серверів та через професійний API `SendGrid` з DKIM-підписами, що гарантує 100% доставку рахунків прямо у вхідні батьків.
